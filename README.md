# Finanzas-Personales-App-de-Gesti-n-de-Ingresos-y-Gastos
Aplicación moderna para gestionar tus ingresos y gastos con fluidez. Backend en Flask y frontend en Tkinter, diseñada para ofrecer control financiero claro, rápido y visualmente atractivo. Ideal para usuarios que necesitan orden, reportes y balance en un solo lugar.
